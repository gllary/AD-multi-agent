# Cohort D Retained Raw Inputs

This folder contains de-identified retained Cohort D checkpoint input tables for the manuscript analysis denominator (n = 1,010).

Files:

- `cohort_D_CP1_demo_history_exam.csv`: CP1 demographics, history, and examination fields.
- `cohort_D_CP2_demo_history_exam_lab.csv`: CP2 demographics, history, examination, and laboratory fields.
- `cohort_D_CP2E_demo_history_exam_lab_echo.csv`: CP2E/CP4-style demographics, history, examination, laboratory, and echocardiography fields.

These are filtered to `../derived/cohort_D/RETAINED_sample_ids.csv` and therefore match the current Cohort D retained analysis set described in the manuscript.
